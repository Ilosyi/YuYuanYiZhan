Place the default images for marketplace listings in this folder.

Required file names:
- sale.jpg        (默认用于出售模块)
- acquire.jpg     (默认用于收购模块)
- help.jpg        (默认用于帮帮忙模块)
- lostfound.jpg   (默认用于失物招领模块)

You can replace these files with any images you like. The application will serve them directly from `/default-images/<name>.jpg`.
